Tehnologie de Ultimă Generație

Cum Funcționează

  1. Capturare Video: Camera USB captează imagini în timp real
  2. Detectare MegaDetector: AI-ul identifică animalele cu precizie ridicată
  3. Clasificare Specii: Rețea neurală specializată determină specia de urs
  4. Validare Temporală: Confirmă detecția prin analiza mai multor cadre
  5. Alertă Audio: Declanșează sunet specific pentru specia detectată

Instalare Rapidă

Cerințe Sistem

  • Python 3.8+
  • Camera USB
  • Windows 10/11, Linux sau macOS
  • 4GB RAM minim (8GB recomandat)
  • GPU NVIDIA (opțional, pentru performanță)

Pași de Instalare

  1. Clonează repository-ul: 
    git clone https://github.com/yourusername/ursa_major_bear_detector.git
cd ursa_major_bear_detector
  2. Instalează dependențele: 
    pip install -r requirements.txt
  3. Configurează sistemul: 
    cp config.example.yaml config.yaml
# Editează config.yaml pentru setările tale
  4. Rulează aplicația: 
    python main.py --config config.yaml
